### A Simple QtVirtualKeyBoard 

This simple virtual keyboard is designed to address the issue where client-application cannot invoke the Qt virtual keyboard in a Weston environment.

My issue is very similar to this  below:
https://bugreports.qt.io/browse/QTBUG-94994
